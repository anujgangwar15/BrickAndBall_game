# 🧱 Brick And Ball Game

A classic 2D **Brick Breaker Arcade Game** built entirely in **Java** using **Swing** and **AWT**. The game recreates the nostalgic arcade experience where players control a paddle to bounce a ball and destroy bricks while earning points.

## 🎮 Features

- 🏓 Smooth paddle movement using keyboard controls
- ⚡ Real-time ball physics and collision detection
- 🧱 Dynamic brick-breaking mechanics
- 📊 Score tracking system
- 🎯 Win and Game Over conditions
- 🚀 Built using pure Java (Swing & AWT)
- 💻 Lightweight desktop application
- 🔄 Restart game functionality

## 🛠️ Technologies Used

- Java
- Swing
- AWT
- Object-Oriented Programming (OOP)

## 🎯 How to Play

1. Launch the game.
2. Use the **Left Arrow (←)** and **Right Arrow (→)** keys to move the paddle.
3. Prevent the ball from falling below the paddle.
4. Break all the bricks to win the game.
5. Press **Enter** to restart after winning or losing.

## 🚀 Getting Started

### Prerequisites

- Java JDK 8 or higher
- Any Java IDE (VS Code, IntelliJ IDEA, Eclipse, NetBeans)

### Clone the Repository

```bash
git clone https://github.com/anujgangwar15/BrickAndBall_game.git
cd BrickAndBall_game
```

### Compile and Run

```bash
javac Main.java
java Main
```
